# Project Structure

## Root

- `server.js` - main HTTP server and UI rendering logic
- `start.sh` - startup helper script
- `package.json` - project metadata and scripts
- `.gitignore` - excludes runtime data and secrets
- `README.md` - project overview

## Runtime-only directories (not committed)

- `data/` - local auth config and initial password
- `logs/` - runtime logs
- `certs/` - local certificates
