# sysinfo_panel

一个轻量的系统信息面板，基于 Node.js 原生 HTTP 服务实现。

## 项目结构

- `server.js`：主服务与页面渲染
- `start.sh`：启动脚本
- `package.json`：项目元信息与脚本
- `docs/PROJECT.md`：项目结构说明
- `data/`：本地配置（默认不提交）
- `logs/`：运行日志（默认不提交）
- `certs/`：证书目录（默认不提交）

## 当前特性

- 登录鉴权
- 系统概览
- 网络接口展示
- 磁盘使用展示
- 监听端口展示
- Docker 容器页
- systemd 服务状态页
- 告警中心
- 自动刷新
- 自定义刷新时间
- 右下角浮动时间盒子
- 基础移动端 / 平板端适配
- 赛博风界面与动效

## 启动

```bash
node server.js
```

或：

```bash
bash ./start.sh
```

## 默认监听

- `127.0.0.1:18888`
- 基础路径：`/sysinfo`

## 说明

仓库默认不包含：

- 运行日志
- 本地配置
- 密码文件
- 证书文件
- 其他运行期敏感数据
